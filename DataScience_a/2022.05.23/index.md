# 有理数とは
- 分数の形で表せられる数
  - a/b
  - ここでa, bは整数
  - 互いに素
    - 素とは... 1以外に約数がない

- a²は3の倍数. したがって、aも3の倍数 (aは整数)
  - a² = 3m (ここで、mは整数, 整数であれば何でも良い)
  - a = √3m -> 整数にしたい
    - (3m ≒ ( X )²)
    - X = 3*3n² (ここでnは整数)
    - m = 3n²
  - a = √3*3n²
    - a = √(3n)²
    - a = 3n (nは整数)

- a² = 3m(a²は3の倍数) --> a = 3n(aも3の倍数)
