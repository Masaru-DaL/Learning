# 有理数とは
- 分数の形で表せられる数
  - a/b
  - ここでa, bは整数
  - 互いに素
    - 素とは... 1以外に約数がない

- a²は3の倍数. したがって、aも3の倍数 (aは整数)
  - a² = 3m (ここで、mは整数, 整数であれば何でも良い)
  - a = √3m -> 整数にしたい
    - (3m ≒ ( X )²)
      - √3m = √3 3 n n
      - √3m = √3 3n²
      - √3m = √ (3n)²
      - a = √(3n)²
      - a = 3n

- a² = 3m(a²は3の倍数) --> a = 3n(aも3の倍数)

## 問1
a²が2の倍数ならば、aも2の倍数を示せ！

- a² = 2m
- a = √2m
  - 2m ≒ (X)² // √を外す為に二乗の形にしたい
  - 2m = 2*2, n*n
  - 2m = 2*2n²
  - 2m = (2n)²
- = √(2n)²
- a = 2n
