* f'(x) -> 微分
  + 傾き = (縦軸の変化量) / (横軸の変化量)

f(x) = x^2

- 公式
$f(x) = ax^n$
$f'(x) = nax^{n-1}$

- 導関数
$f'(x) = \frac{lim}{⊿x->0}$

$\frac{(x+⊿x) - f(x)}{⊿x}$

- 問題
  - $f(x) = x^2$の時

1. $\frac{lim}{⊿x->0}$ = $\frac{(x+⊿x)^2 - x^2}{⊿x}$

2. $\frac{lim}{⊿x->0}$ = $(a + b)^2$
-> $a^2 + 2ab + b^2$

$\frac{x^2 + 2x⊿x + ⊿x^2 - x^2}{⊿x}$

3. $\frac{lim}{⊿x->0}$ = $\frac{2x⊿x + ⊿x^2}{⊿x}$

4. $\frac{lim}{⊿x->0}$ = $2x + ⊿x$

- 微分の公式を用いる
$f(x) = ax^n$　-> $f'(x) = nax^{n-1}$

- 導関数を使うと

$f(x) = ax^n$　-> 導関数を使う ->  $f'(x) = nax^{n-1}$
