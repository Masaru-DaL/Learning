# 確率分布
## 離散型確率分布 / 連続型確率分布
- 微分
  - 傾き
- 積分
  - 面積
同じ関数であれば微分と積分は戻せる

- 正規分布
  - 平均値で左右対称

## 確率分布とは？
どのような値を、どのような確率でとるかは、一定の法則・ルールにより定まっている。
その法則を確率分布と呼ぶ

### 連続型確率分布
- x上の平均点を境に左右対称に引かれた線 => 確率密度関数
  - y=f(x)

- y ≒ 確率, y = 関数

- 確率 = 面積, 1以下

## 正規分布
μ: 平均(値)
σ: 標準偏差(=分散(ばらつき))
上記2つが分かると、正規分布の形が分かる。
※μ, σ = パラメータ

### より正確には
Pr(μ - σ ≤ X ≤ μ ＋ σ) ~ 0.6827 // 68%と覚えて良い
Pr(μ - 2σ ≤ X ≤ μ ＋ 2σ) ~ 0.9545 // 95%と覚えて良い
Pr(μ - 3σ ≤ X ≤ μ ＋ 3σ) ~ 0.9973 // 99.7%と覚えて良い
