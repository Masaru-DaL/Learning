# 確率
## 正規分布の形状
連続の値を取りたい！
連続の値の入力方法
- 平均値(μ)
- 標準偏差(σ)

### Excel
- 連続の値入力
1. ホーム
2. ↓フィル(Σの下にある)
3. 連続データの作成
4. セッティング
   1. 範囲: 列
   2. 種類: 加算
   3. 増加値: 1
   4. 停止値: +3σの値

#### 名前ボックスの活用
ExcelはCopy&Pasteが得意！

1. 何列まで数値が入っているかをチェックする
   1. Ctrl +↓キー1 (or Ctrl + Shift + ↓キー)
   2. 名前ボックスで何行まであるかチェックする
2. コピーしたい場所をコピー状態にする
3. 名前ボックスに入力
   1. コピー状態の場所:コピーを終わらせたい場所
   2. 例: D12:D72
4. エンター
5. Ctrl + V
